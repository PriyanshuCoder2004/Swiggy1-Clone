# 🍔 Swiggy Clone – Food Ordering Web App

A modern **Swiggy Clone** built using **React.js** and **Tailwind CSS** that replicates the core functionality of the Swiggy food delivery platform.
This project allows users to explore restaurants, browse menus, search food items, and manage their cart with a smooth and responsive user interface.

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/PriyanshuCoder2004/Swiggy1-Clone.git
```

Go to the project directory:

```
cd Swiggy1-Clone
```

Install dependencies:

```
npm install
```

Run the development server:

```
npx parcel src/index.html
```
